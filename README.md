# アイ・ラブ・ユー・つ～
プレイ：https://unityroom.com/games/i_love_you_too

## ジャンル
つむぎ つなげて つくりだす
「パズルアクションゲーム」

## ゲーム紹介
ひらがなの「つ」になって、
飛んでくる文字を避けながら、
必要な言葉を順番に集めるゲームです。

違うひらがなに当たっても、
2回までなら「つ」は「たえる」ことができます！
言葉を組み立てて彼の気持ちを「つたえて」下さい！！

※Safariだと動作しないかもです。　推奨:Chrome
